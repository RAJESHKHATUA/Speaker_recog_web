Speaker Recognition System

How It Works

The Voice Enrollment System allows users to securely authenticate using their voice. Simply record your voice by reading a set of phrases, and the system will create a unique voiceprint for you.

Purpose

This project is designed to provide a secure and convenient way to authenticate users using biometric voice recognition. It can be used for attendance tracking, secure logins, and more.
